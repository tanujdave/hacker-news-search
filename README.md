## Hacker News Search
### Prerequisites

* node
* npm

1. Clone this repository

2. Install client dependencies
    ```bash
    $ cd client
    $ npm install
    ```

## Run the app

3. Start the client
    ```bash
    $ cd client
    $ npm start
    ```
4. Browse to `http://localhost:3000/`

5. Preview
